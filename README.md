# Warbridge
https://www.spigotmc.org/resources/warbridge.134284
## ENGLISH

Warbridge is a fully configurable Warbridge minigame plugin designed for BungeeCord / Waterfall networks.

Unlike traditional multi-room plugins, Warbridge uses a **one-server-per-game architecture**, meaning each server instance represents a single match.

### Features

- One-server-per-game architecture
- Fully configurable system
- Edit mode for map setup
- Multi-team support (2 / 4 teams)
- Round-based gameplay
- Cage start system
- Portal scoring system
- Team selection GUI
- Spectator mode
- Dynamic scoreboard (board.yml)
- Dynamic MOTD system
- Bilingual support (English & Chinese)
- PlaceholderAPI support
- Advanced announcement system
- Fully configurable sounds, menus and messages

### Installation

1. Install the plugin on Spigot/Paper (1.8.8–1.8.9 recommended)
2. (Optional) Install PlaceholderAPI
3. Start the server
4. Run:
   /warbridge edit
5. Configure the map
6. Run:
   /warbridge setup finish
7. Restart the server

## 中文说明
https://www.spigotmc.org/resources/warbridge.134284
Warbridge 是一个为 BungeeCord / Waterfall 群组服务器设计的战桥小游戏插件。

本插件采用 **一服一局** 架构，每个子服即一场独立游戏。

### 功能特点

- 一服一局架构
- 全部功能可配置
- 编辑模式快速搭建地图
- 支持多队伍（2队 / 4队）
- 回合制玩法
- 笼子开局机制
- 传送门得分系统
- 队伍选择菜单
- 观战模式
- 动态计分板
- 动态 MOTD
- 中英双语支持
- PlaceholderAPI 支持
- 高级播报系统
- 全配置化音效与菜单

### 安装方法

1. 将插件放入 plugins 文件夹
2. 启动服务器
3. 执行：
   /warbridge edit
4. 配置地图
5. 执行：
   /warbridge setup finish
6. 重启服务器
